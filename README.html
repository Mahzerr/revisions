Chap. 11
Transformations nucléaires
Noyaux isotopes
Noyaux possédant le même nombre de protons (Z) mais des nombres de neutrons (N) différents. Ils appartiennent au même élément chimique.
Exemples
→Carbone : ¹²₆C · ¹³₆C · ¹⁴₆C
→Néon : ²⁰Ne (90,48%) · ²¹Ne · ²²Ne
→Silicium ²⁸Si (Z=14, N=14) et ²⁹Si (Z=14, N=15)
Lois de conservation
Lors d'une transformation nucléaire, deux grandeurs sont toujours conservées :
Conservation
A₁ + A₂ = A₃ + A₄ (nucléons)
Z₁ + Z₂ = Z₃ + Z₄ (charges)
Exemples
→¹⁴₆C → ¹⁴₇N + ⁰₋₁e
masses : 14=14+0 ✓ charges : 6=7+(-1) ✓
Radioactivité β⁻
émission e⁻
Transformation d'un neutron en proton avec émission d'un électron (⁰₋₁e). Concerne les noyaux avec excès de neutrons.
Exemples
→Carbone 14 :¹⁴₆C → ¹⁴₇N + ⁰₋₁e
→Phosphore 32 :³²₁₅P → ³²₁₆S + ⁰₋₁e
Radioactivité β⁺
émission e⁺
Transformation d'un proton en neutron avec émission d'un positon (⁰₊₁e). Concerne les noyaux avec excès de protons.
Exemples
→Fluor 18 :¹⁸₉F → ¹⁸₈O + ⁰₊₁e(imagerie TEP)
→Césium 130 :¹³⁰₅₅Cs → ¹³⁰₅₄Xe + ⁰₊₁e
Radioactivité α
émission ⁴He
Émission d'un noyau d'hélium (⁴₂He). Concerne les noyaux lourds avec excès de nucléons.
Exemples
→Uranium 238 :²³⁸₉₂U → ²³⁴₉₀Th + ⁴₂He
→Radium 226 :²²⁶₈₈Ra → ²²²₈₆Rn + ⁴₂He
Fission nucléaire
Un noyau lourd se scinde en 2 noyaux plus légers sous l'impact d'un neutron. Libère une énergie colossale.
Exemples
→Uranium 235 :²³⁵₉₂U + ¹₀n → ⁹²₃₆Kr + ¹⁴¹₅₆Ba + 3¹₀n
→1g d'U-235 libère 7,3×10¹⁰ J (= fusion de 220t de glace)
Fusion nucléaire
Deux noyaux légers s'assemblent pour former un noyau plus lourd. Se produit dans les étoiles. Projet ITER.
Exemples
→Deutérium + Tritium :²₁H + ³₁H → ⁴₂He + ¹₀n
→Dans les étoiles :⁸₄Be + ⁴₂He → ¹²₆C

Chap. 12
Transformations physiques
Changement d'état
Lors d'un changement d'état, les entités chimiques et espèces chimiques sont conservées. C'est une transformation physique.
ÉtatCaractère
SolideCondensé + ordonné
LiquideCondensé + désordonné
GazeuxDispersé + désordonné
Condensé vs Dispersé
État condensé : la distance entre entités est du même ordre de grandeur que leur taille.
État dispersé : la distance entre entités est grande devant leur taille.
Écriture symbolique
→Al₂O₃(s) → Al₂O₃(l) fusion alumine
→H₂O(g) → H₂O(l) liquéfaction eau
→I₂(s) → I₂(g) sublimation diiode
Corps pur & T° constante
Pour un corps pur, le changement d'état se fait à température constante (pression constante). Ces températures permettent d'identifier une espèce chimique.
Propriétés clés
→θfusion = θsolidification
→θvaporisation = θliquéfaction
→Palier de température = coexistence de 2 états
Fusion vs Dissolution
FusionDissolution
Espèces1 seule≥ 2 (solvant + soluté)
RésultatMême espèce, état liquideSolution homogène
TypePhysiquePhysique/chimique
La dissolution nécessite un solvant — espèce chimique particulière qui conduit à une solution.
Endo / Exothermique
Endothermique : le système reçoit de l'énergie par transfert thermique depuis l'extérieur.
Exothermique : le système cède de l'énergie par transfert thermique vers l'extérieur.
EndothermiqueExothermique
Fusion, vaporisation, sublimationSolidification, condensation, liquéfaction
Énergie massique l
Formule clé
l = Q / m ⟹ Q = m × l

Q : énergie (J)
m : masse du système (kg)
l : énergie massique (J/kg)
Exemples
→Fusion glace : lfus=334 kJ/kg ; m=450gQ = 334×10³ × 0,45 = 1,50×10⁵ J
→Vaporisation eau : lvap=2511 kJ/kg ; Q=130 kJm = 130/2511 = 5,2×10⁻² kg
Chap. 13
Transformations chimiques
Transformation chimique
Un système évolue d'un état initial à un état final avec consommation de certaines espèces (réactifs) et formation de nouvelles espèces (produits).
Exemples
→Combustion méthane : CH₄ + O₂ → CO₂ + H₂O
→Corrosion par acide :Fe + H⁺ → Fe²⁺ + H₂
Équation de réaction
L'équation doit être équilibrée pour respecter la conservation des éléments chimiques et de la charge électrique. On utilise des nombres stœchiométriques.
Règles d'équilibrage
1. Équilibrer C, puis H, puis O
2. Vérifier la charge électrique
Exemples équilibrés
→Éthanol :C₂H₆O + 3O₂ → 2CO₂ + 3H₂O
→Propane :C₃H₈ + 5O₂ → 3CO₂ + 4H₂O
→Aluminium :2Al + 6H⁺ → 2Al³⁺ + 3H₂
Réactif limitant
Le réactif limitant est entièrement consommé lors d'une transformation totale. Les autres réactifs sont en excès. Sa disparition arrête la réaction.
Identification — réaction aA + bB → ...
Si n(A)/a < n(B)/b ⟹ A est limitant
Si n(A)/a > n(B)/b ⟹ B est limitant
Méthode
→Comparer les quotients n(réactif)/coefficient stœchio
→Le plus petit quotient → réactif limitant
